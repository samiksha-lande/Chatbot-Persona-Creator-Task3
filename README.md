# Custom Chatbot Persona Creator - Task 3

## Project Goal
To design and document three distinct, prompt-powered chatbot personalities. This project demonstrates skills in advanced prompt engineering, tone control, persona scripting, and basic chatbot UX design.

## Skills Gained
* Prompt Engineering (Defining System Prompts and Guardrails)
* Chatbot User Experience (UX)
* Tone Control and Persona Consistency
* Persona Scripting and Conversation Flow design

## Deliverables
1.  **3 Chatbot Persona Documents:** Completed instruction sheets for three roles.
2.  **Sample Conversation Flows:** Included within the persona documents and demo.
3.  **1 Working Chatbot Demo:** A documented conversation flow for the chosen persona (Samiksha).
4.  **Prompt Documentation:** Finalized System Prompts for all three personas.

---

## The Three Personas Designed

| Role | Name | Core Function | Core Tone |
| :--- | :--- | :--- | :--- |
| Wellness Coach | Samiksha | Non-medical, encouraging advice on fitness and stress. | Gentle, encouraging. |
| Coding Tutor | Professor Jin | Structured explanations of Python and AI concepts. | Technical, precise. |
| Summarizer | Archivist AI | Formal summarization of text into three bullet points. | Formal, purely factual. |
